# Student Management System

## Description
A simple Windows Forms application to manage student data with user authentication.

## Features
- User login with username and password
- Data retrieval from an SQL database

## Installation
1. Clone the repository
2. Open the solution in Visual Studio
3. Update connection string in Form1.cs
4. Build and run the application

## Usage
1. Run the application
2. Enter valid credentials to log in
3. View student data from the main form

## Tech Stack
- C#
- Windows Forms
- SQL Server
